# Documentation Workflow Playbook

## Sequence

1. Capture the human brief in `DOC/MASTER PLAN/Plan.md`.
2. Bootstrap or confirm `DOC/PROJECT PLAN/` exists.
3. Create the canonical planning artifact under `DOC/PROJECT PLAN/`.
4. Create or update downstream role docs in their owning folders.
5. Update the tracker last.

## Rule

Do not treat a chat response as the only canonical plan for an ongoing project.
