# Startup Conversion

## Voice
- Direct, benefit-first, urgency-aware.
- One offer per page, ruthless about a single CTA.
- Social proof loud and early.

## Density
- Hero spacious and assertive.
- Below the fold: medium density, scrollable proof rail, FAQ, final CTA.

## Color Direction
- One bold brand hue + one high-energy accent.
- Strong contrast between CTA and background; CTA must be the brightest element on screen.

### Default token roles
- background:  off-white or near-black depending on brand mood
- surface:     white (light) / deep neutral (dark)
- primary:     bold saturated brand hue
- accent:      vivid contrast (electric cyan/lime/coral)
- destructive/success/warning/info: standard semantic palette

## Typography
- Display family: bold geometric sans with high contrast weights.
- Body family: humanist sans.
- Display scale: 80 / 72 / 64 / 56 (assertive).
- Heading scale: 40 / 32 / 28 / 24.
- Body scale: 18 / 16.
- Line-height: 1.0–1.1 display, 1.2 headings, 1.6 body.
- Use weight contrast: 800/900 on display, 400/500 body.

## Spacing
- Section rhythm: 64–96px desktop / 48–64px tablet / 40–48px mobile.

## Surface system
- Page base: flat brand-tinted neutral.
- Elevated: borderless cards with shadow, or hairline-bordered cards with no shadow.
- Inset: testimonial cells, code/feature cells.
- Overlay: solid translucent.

## Radius
- Cards 12, hero panels 16, inputs/buttons 10–12, pill-shaped CTA optional.

## Motion personality
- Slightly energetic, fast, purposeful.
- Macro motion 180–220ms; micro 100–160ms.
- Spring on chip selection, button press.
- Hover scale 1.03; press 0.97.
- Hero entrance: stagger reveal of headline / sub / CTA.

## Imagery direction
- Real product screenshots, founder photos, customer logos.
- Avoid stock illustrations.
- Bold supporting graphics (oversized arrows, underlines, hand-drawn feel).

## Hero composition
- Headline + sub + dual CTA (primary + "watch demo / read more").
- Below: customer logo strip → product UI panel → testimonial cluster.

## CTA hierarchy
- ONE primary CTA repeated in hero, mid-page, footer.
- Secondary CTA only if it does not compete with primary (e.g., "watch 60s demo").

## Iconography
- Mixed: outline for utility, filled for emphasis.
- Sizes 20/24/32.

## Forbidden in this archetype
- More than one offer per page.
- Hero clutter; CTA must be unmistakable.
- Endless gradient walls; one accent gradient max.
