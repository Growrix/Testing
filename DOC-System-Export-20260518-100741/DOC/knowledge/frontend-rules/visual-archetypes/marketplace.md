# Marketplace

## Voice
- Neutral-platform tone.
- Facilitator, not opinionated.
- Buyer-side: discovery and trust.
- Seller-side: clarity and control.

## Density
- Hero medium.
- Listings dense (high results-per-screen).
- Detail page balanced with rich media + specs + reviews.

## Color Direction
- Neutral-led to let listings shine.
- Two surface colors: search/filter rail and result canvas.
- Trust signals (verified, top-rated) get a single distinctive color.

### Default token roles
- background:  off-white
- surface:     white
- inset:       light grey (filter rail, sticky bars)
- primary:     trust hue (deep teal/blue/green)
- accent:      conversion hue ("Buy", "Book")
- destructive/success/warning/info: standard semantic palette
- verified:    high-trust hue (e.g., teal-green)

## Typography
- Display family: humanist sans, friendly.
- Body family: same family.
- Heading scale: 32 / 28 / 24 / 20 / 18 / 16.
- Body scale: 16 / 14 / 13.
- Tabular figures for prices.

## Spacing
- 8px base.
- Listing card padding 16–20; sticky filter rail 16; gallery 24.

## Surface system
- Page base: off-white.
- Elevated: card with hairline border and mild shadow.
- Inset: filter rail and sort/sticky bars.
- Overlay: translucent for quick-view modals.

## Radius
- Cards 12, hero panels 16, inputs 8–10.

## Motion personality
- Snappy, functional.
- Macro 160–220ms; micro 100–160ms.
- Hover lift on listing cards 1.01–1.02 with shadow shift.
- Filter chip selection: spring.

## Imagery direction
- Listing-driven; aspect ratios standardized per listing kind.
- Verified-badge overlays where applicable.

## Hero composition
- Search bar + category chips first.
- Featured listings band below.
- Trust strip (verification, reviews, support).

## CTA hierarchy
- Buyer side: "Search", "Buy", "Book", "Request quote".
- Seller side: "List item", "Manage listings", "View earnings".

## Iconography
- Outline 1.5px; sizes 16/20/24.
- Filled for selected category chip and verified badge.

## Trust real estate
- Verification badge inline on every seller mention.
- Review aggregate next to listing title.
- Dispute / safety policy links in footer and at checkout.

## Mobile rules
- Bottom dock: Search, Saved, List, Messages, Account.
- Filter and sort live in bottom sheets.

## Forbidden in this archetype
- Single-side bias (forgetting seller-side surfaces).
- Hover-only listing detail.
- Hidden disclosures (fees, refund policy) — must be inline near the price.
