# Editorial Premium

## Voice
- Refined, calm, high-trust.
- Treats whitespace as content.
- Privileges typographic hierarchy over decoration.

## Density
- Spacious on hero, value, and proof sections.
- Balanced on listings and detail.
- Dense only when data display is unavoidable.

## Color Direction
- Light-first.
- Warm-neutral background with controlled chromatic accent.
- Accent saturation kept under 12% of total screen area at any moment.
- Two-color discipline: one primary, one accent; no third hue without justification.

### Default token roles
- background: warm paper (e.g., `#F5F1E8`)
- surface:    near-white (e.g., `#FFFDF8`)
- inset:      slightly darker neutral
- border:     low-contrast neutral
- text:       deep graphite
- muted:      slate-grey
- primary:    deep teal or deep navy
- accent:     warm copper / amber
- destructive: muted red
- success:    forest green
- info:       muted blue
- warning:    amber

## Typography
- Display family: a refined geometric sans (e.g., Cabinet Grotesk class).
- Body family: a humanist sans (e.g., Manrope class).
- Mono family: technical-feel mono (e.g., JetBrains Mono class).
- Display scale: 72 / 64 / 56 / 48.
- Heading scale: 40 / 32 / 28 / 24 / 20 / 18.
- Body scale: 18 / 16 / 14.
- Line-height: 1.05–1.15 display, 1.2–1.35 headings, 1.55–1.7 body.
- Letter-spacing: -1% display, 0% body, +6% overline/eyebrow.

## Spacing
- Section rhythm: 96px desktop / 64px tablet / 48px mobile.
- Card padding: 24 standard / 32 premium / 16 dense mobile.

## Surface system
- Page base: warm paper with subtle grid noise.
- Elevated: soft white, hairline border, mild shadow.
- Inset: slightly darker neutral for filters and code wells.
- Overlay: blurred dark translucent.

## Radius
- Cards 16, hero panels 24, inputs/buttons 12.

## Motion personality
- Subtle, slow, controlled.
- Macro motion duration band 200–260ms.
- Micro-animation 120–180ms.
- Easing default: `cubic-bezier(0.22, 1, 0.36, 1)`.
- Hover scale 1.01–1.02; never 1.05+.

## Imagery direction
- Editorial photography, real product UI, annotated mockups.
- Avoid generic developer stock.
- Aspect ratios: 16:10 panels, 4:3 features, 9:16 mobile.

## Hero composition
- Layered: paper base, frosted panel, inset data well, elevated CTA card.
- Two-column on desktop with media panel; single-column on mobile.

## CTA hierarchy
- One primary, one secondary; never three.
- Primary: filled, brand primary, medium-large.
- Secondary: outline or ghost.

## Iconography
- Outline-first, 1.75px stroke, sizes 16/20/24/32.
- Filled icons reserved for status badges.

## Forbidden in this archetype
- Neon gradients, oversaturated colors, busy patterns.
- Decorative motion that does not aid clarity.
- More than two accent hues at once.
