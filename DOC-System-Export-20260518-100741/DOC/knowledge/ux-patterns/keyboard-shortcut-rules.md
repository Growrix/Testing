# Keyboard Shortcut Rules

## Purpose
Consistent shortcut behavior across product surfaces.

## Baseline shortcuts
- Cmd/Ctrl+K: command palette
- g then h: go home
- /: focus search (when present)
- Esc: close modal or dismiss overlay

## Rules
- Shortcut map must be declared per page and globally de-duplicated
- Do not shadow browser/system critical shortcuts
- Provide shortcut hints in menus and tooltips
- Support user-disabled shortcuts for accessibility needs

## Anti-patterns
- Inconsistent bindings for same action across pages
- Hidden shortcuts with no discoverability path
