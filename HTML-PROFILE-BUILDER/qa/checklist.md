# QA Checklist

## Visual Check
- [ ] Open the HTML file in Chrome.
- [ ] Check widths at 360px, 375px, and 414px.
- [ ] Confirm there is no horizontal overflow.
- [ ] Confirm the hero, services, contact, and CTA remain readable.

## Content Check
- [ ] Business name matches the brief.
- [ ] Tagline matches the brief.
- [ ] All services and prices match the brief.
- [ ] Hours match the brief.
- [ ] Contact details and links match the brief.
- [ ] No invented facts or placeholder text appear.

## Link Check
- [ ] Phone link opens tel:.
- [ ] WhatsApp link opens wa.me.
- [ ] Map link opens the provided Google Maps URL.
- [ ] Social links open the provided social pages.

## Approval
- [ ] QA reviewer name filled.
- [ ] Approval timestamp filled.
- [ ] Client-ready notes recorded.
