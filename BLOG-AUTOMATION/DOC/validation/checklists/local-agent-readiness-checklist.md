# Local Agent Readiness Checklist

- Public local wrappers exist under `.github/agents/`.
- Canonical local agent sources exist under `DOC/agents/`.
- Local registry documentation is updated.
- Supporting specs and checklists exist for non-trivial local agents.
- The core agent exists and can route whole-system tasks to specialist local agents.
- Local agent changes do not introduce workspace-root mixing.
- Remaining gaps are documented explicitly.
